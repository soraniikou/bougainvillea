🪽誰かに伝える必要のない、自分だけの願いを「ゆめくい」に預けます。
​「今日は、誰とも目が合わなくてもいい」「一言だけ、自分のために笑いたい」
​言葉をアプリに書き込むと、それは空に溶けていき、夜には静かな星となって、次の日の朝の「希望の種」に変わる。
​「好き」にならなくていい、という全肯定


対人でエネルギーを使い果たしてしまうこともありますよね。
「人を好きにならなきゃ」と思うと苦しいですが「自分という静かな場所を守りながら、他人とすれ違うだけ」でいいと思えたら、それだけで心はぐっと軽くなります。
​ブーゲンビリア💐の花は、中心にある小さくて白い部分が本当の花で、色鮮やかな周りは「苞（ほう）」、つまり葉が変化したものです。
「自分を守るための色鮮やかなバリア（苞）があってもいい。その中心で、自分だけの白く小さな希望が咲いていればいい」

対人関係が得意ではないが、不安は生きている証拠
と思えるようにアプリ作りました
